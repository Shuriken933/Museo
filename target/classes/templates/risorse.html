Risorse usate

Qui segue un elenco delle risorse utilizzate per l'implementazione di questo sistema informativo

Front-end

Bootstrap 5.0

Bootstrap è un framework per la costruzione di siti responsive (design per mobile, tablet e desktop) che offre file css e javascript tramite i quali si ha elementi già stilizzati e pronti per la maggior parte degli utilizzi. Nel nostro contesto abbiamo usato il menu, il sistema delle griglie (flexbox) e le classi in grado di rendere il sito responsive.

JQuery 3.6.0

JQuery è una libreria javascript che permette l'uso di codice semplificato per la gestione di animazioni, eventi (azioni con il mouse) ecc. Per questa piattaforma è stato utilizzato per la realizzazione dello slider, nel quale necessitavamo di aggiungere e rimuovere classi css (funzioni "remove" e "add").

Font ~ Cormorant Garamond

Questo font è stato impiegato per la stilizzazione dei titoli (h1, h2, h3, h4, h5, h6).

Font ~ Poppins

Questo font è stato impiegato per la stilizzazione dei paragrafi e delle voci dei menu.

Back-end

Thymeleaf

Thymeleaf è un moderno motore di template Java lato server per ambienti web e standalone.

Spring Boot 2.5.0

Spring Framework è un framework applicativo e un contenitore di controllo di inversione per la piattaforma Java. Le funzionalità principali del framework possono essere utilizzate da qualsiasi applicazione Java, ma esistono estensioni per la creazione di applicazioni Web su Java EE
